# 🧬 ACTG175 HIV Clinical Trial Survival Analysis

## Overview

This project analyzes the ACTG 175 HIV Clinical Trial dataset using R. The objective is to evaluate the effectiveness of different antiretroviral treatment regimens and identify clinical factors associated with HIV disease progression using survival analysis and statistical modeling techniques.

The project includes:

- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Kaplan–Meier Survival Analysis
- Log-Rank Testing
- Cox Proportional Hazards Modeling
- Logistic Regression
- Data Visualization

---

## Dataset Information

- Dataset: ACTG 175 HIV Clinical Trial Study
- Total Patients: 2,139
- Clinical Variables: 24
- Outcome: Time to AIDS-defining event or death

---

## Project Objectives

- Analyze survival outcomes of HIV patients.
- Compare treatment effectiveness across treatment groups.
- Identify significant prognostic factors affecting disease progression.
- Apply survival analysis techniques to clinical trial data.
- Generate clinically interpretable visualizations and insights.

---

## Key Findings

- Survival outcomes differed across treatment groups.
- Combination therapy demonstrated improved survival compared with monotherapy.
- Higher baseline CD4 counts were associated with better outcomes.
- Symptomatic patients showed increased risk of disease progression.
- Cox modeling identified significant predictors of survival.
